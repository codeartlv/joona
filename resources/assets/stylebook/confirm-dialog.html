Confirm dialog

A component that can be used to ask user to confirm various things. Can be used either directly on link or as standalone component. Dialog automatically closes when any of the buttons is clicked.

Using as standalone component


let caption = 'Confirm deletion';
let message = 'The records will be deleted. Proceed?';
let buttons = [
	{
		caption:'OK',
		disabled:false,
		role:'primary',
		callback : () => {
			// code to be executed
		}
	},
	{
		caption:'Cancel',
		disabled:false,
		role:'secondary',
		callback : () => {
			// code to be executed
		}
	}
];
let confirmDialog = new ConfirmDialog(
	caption, message, buttons
);

confirmDialog.open();
